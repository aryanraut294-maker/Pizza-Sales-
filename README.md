# Pizza-Sales-
## 🍕 Pizza Sales Dashboard
Project Description

The Pizza Sales Dashboard is an end-to-end data analytics project developed using SQL and Power BI. The project analyzes pizza sales data to transform raw transactional records into meaningful business insights through interactive dashboards. It enables stakeholders to monitor sales performance, customer purchasing behavior, and product performance, supporting better business decisions.

## Project Purpose

The main purpose of this project is to analyze pizza sales data and answer key business questions that can help improve sales performance and profitability. The dashboard provides an easy-to-understand visualization of important KPIs, sales trends, customer behavior, and product performance.

## Business Problems Solved

This project helps answer important business questions such as:

What is the total revenue generated by the business?
How many orders and pizzas were sold?
Which pizzas generate the highest revenue?
Which pizza categories and sizes are most popular?
Which days and hours receive the highest number of orders?
How does revenue change over time?
Which pizzas perform best within each category?
What is the average number of pizzas ordered per customer order?
Which product categories contribute the most to total sales?
